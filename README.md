## CodeMield – Developer Portfolio & Service Website

CodeMield is a **modern responsive developer portfolio and service business website** designed for companies or freelancers who provide **App Development and Website Development services**.

The website works as a **business website, developer portfolio, online CV, and client inquiry platform**.

Built using **HTML, CSS, and JavaScript**, the design focuses on **clean UI, smooth animations, and professional layout**.

---

# Features

• Fully responsive design (mobile, tablet, desktop)
• Modern UI with smooth animations
• Developer CV / profile section
• Services section with animated cards
• Pricing section for development services
• Portfolio showcase for projects
• Client inquiry contact form
• WhatsApp contact button
• Call Now button
• Sticky navigation bar
• Mobile friendly hamburger menu
• Clean beginner-friendly code
